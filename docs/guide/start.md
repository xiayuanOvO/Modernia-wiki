- 使用 「」 包裹的为机器人指令
- 使用 【】 包裹的为玩家可自定义填写的内容

## 介绍

《Modernia》旨在为玩家提供一个高度自由、充满魔法奇幻色彩的文字冒险世界。通过随机化的角色生成和丰富的剧情分支，鼓励玩家探索多样的游戏路径，体验从平凡冒险者到传奇英雄的成长历程。灵感来源于《关于我转生变成史莱姆这档事》，结合文字游戏的低门槛与高想象力，打造一款适合奇幻爱好者的沉浸式体验。
